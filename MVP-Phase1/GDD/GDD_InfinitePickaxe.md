## 1. 프로젝트 개요

### 1-1. 기본 정보

| 항목 | 내용 |
|------|------|
| **게임명 (KO)** | 무한의 곡괭이: 끝없는 채굴 |
| **게임명 (EN)** | Infinite Pickaxe: Endless Mining |
| **장르** | Idle / 방치형 성장 게임<br>채굴 테마 RPG (DPS 성장형) |
| **플랫폼** | 모바일 (Android) |
| **타겟** | 캐주얼 게이머, Idle 게임 유저 |
| **개발 형태** | 1인 개발 + 서버 권위 구조 |

### 1-2. 핵심 한 줄 소개

> **"돌에서 행성까지, 곡괭이 하나로 끝없이 캐 나가는 서버 권위형 Idle 채굴 RPG"**

### 1-3. 핵심 특징

- ✅ **서버 권위 구조**: 모든 로직은 서버에서 시뮬레이션, 클라이언트는 렌더링만
- ✅ **오프라인 보상**: 최대 12시간까지 비접속 중에도 채굴 진행
- ✅ **픽셀아트 스타일**: AI 생성 + 수동 보정을 통한 저해상도 픽셀아트
- ✅ **자유로운 도전**: 광물 선택 자유, 추천 시스템으로 효율 가이드
- ✅ **다중 곡괭이 슬롯**: 여러 곡괭이를 동시에 운용하여 DPS 증폭

## 2. 게임 컨셉 & 세계관

### 2-1. 세계관

플레이어는 **특별한 곡괭이의 주인**이 되어, 곡괭이를 강화하며 점점 더 단단한 자원을 채굴해 나간다.

**성장 단계**:
- **초반**: 작은 동굴에서 돌, 석탄, 구리 같은 기본 광물 채굴
- **중반**: 깊은 광산에서 철, 금 같은 귀금속 발견
- **후반**: 빛나는 보석 광맥, 에메랄드 등 희귀 자원 채굴
- **엔드게임** (추후 확장): 운석, 소행성, 행성까지 채굴 대상 확장

### 2-2. 톤 & 무드

- **초반**: 밝고 캐주얼한 동굴/광산 분위기
- **중반**: 빛나는 광맥, 마법/에너지 광물의 신비로운 느낌
- **후반**: 우주적 스케일, 행성 벨트, 성운 등 웅장한 연출 (추후 확장)

**폭력성 없음**: 전투나 살상이 아닌 **자원 채굴·성장·확장**에 초점

## 3. 코어 게임플레이 루프

```
┌─────────────────────────────────────┐
│  1. 광물 선택 (추천 시스템 참고)     │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  2. 자동 채굴 (곡괭이 DPS로 광물 HP 감소) │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  3. 채굴 완료 → 골드/크리스탈 획득   │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  4. 골드로 곡괭이 강화 (DPS 증가)    │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  5. 더 높은 티어 광물 도전           │
└──────────────┬──────────────────────┘
               ↓
              반복
```

**추가 루프**:
- 일일 미션 완료 → 크리스탈 획득
- 크리스탈로 슬롯 해금 → 다중 곡괭이 배치
- 오프라인 보상으로 비접속 중에도 성장

## 4. 게임 시스템

### 4-1. 곡괭이 시스템

#### 4-1-1. 곡괭이 스탯

| 스탯 | 설명 | 역할 |
|------|------|------|
| **채굴력 (Mining Power)** | 초당 광물 HP를 얼마나 깎는지 | DPS |
| **공격 속도 (Attack Speed)** | 1초당 타격 횟수 | MVP에서는 1.0 고정 |
| **크리티컬 확률** | 치명타 발생 확률 | MVP 제외 (추후 확장) |
| **크리티컬 데미지** | 치명타 피해 배율 | MVP 제외 (추후 확장) |

**최종 DPS 계산**:
```
DPS = 채굴력 × 공격 속도 × (슬롯별 곡괭이 합산)
```

#### 4-1-2. 곡괭이 레벨 & 티어

**5레벨 = 1 진화 사이클**

| 레벨 범위 | 티어 | 곡괭이 외형 | 특징 |
|----------|------|-----------|------|
| **Lv 0-4** | T1 | 나무 곡괭이 | 아웃라인 점진적 증가 (1→4px) |
| **Lv 5** | T1→T2 | 돌 곡괭이 | **진화!** 외형 변경 + 이펙트 |
| **Lv 6-9** | T2 | 돌 곡괭이 | 아웃라인 점진적 증가 |
| **Lv 10** | T2→T3 | 철 곡괭이 | **진화!** |
| **Lv 11-14** | T3 | 철 곡괭이 | 아웃라인 증가 |
| **Lv 15** | T3→T4 | 금 곡괭이 | **진화!** |
| **Lv 16-19** | T4 | 금 곡괭이 | 아웃라인 증가 |
| **Lv 20+** | T5+ | 다이아몬드... | Post-MVP 확장 |

**진화 연출**:
- 레벨 5, 10, 15, 20에서 특수 애니메이션
- 파티클 이펙트 + 사운드
- "진화 완료!" 팝업

#### 4-1-3. 곡괭이 강화

**강화 방식**: 확률형 + 천장 (티어별 기본 확률, 실패 보너스 누적, 상한 100%)

**절차**:
1. 강화 탭 진입
2. 현재 레벨/스탯 확인
3. 골드 소모하여 강화 (서버 승인 필수)
4. 레벨업 시 채굴력 증가

**광고 할인**:
- 강화 탭에서 광고 시청 시 **다음 강화 비용 -25%**
- 1일 최대 3회

**확률형 강화 + 천장**
- 티어별 기본 확률: T1(0~4) 100%, T2(5~9) 95%, T3(10~14) 90%, T4(15~19) 85% (최소 30%)
- 실패 시 보너스: 기본 확률의 10%를 누적, 최종 확률 `min(기본+보너스, 100%)`, 성공 시 보너스 0 초기화
- 예시: 기본 30% → 1회 실패 33% → 2회 36% → 3회 39% ... 상한 100%

### 4-2. 광물 & 채굴 시스템

#### 4-2-1. 광물 속성

각 광물은 다음 속성을 가짐:

| 속성 | 설명 |
|------|------|
| **경도 (HP)** | 채굴해야 할 총 HP |
| **골드 보상** | 채굴 완료 시 지급되는 골드 |
| **추천 레벨** | 효율적으로 채굴 가능한 곡괭이 레벨 |
| **재등장 시간** | 채굴 완료 후 다시 선택 가능까지의 대기 시간 (5초 고정) |
| **바이옴** | 광물이 속한 배경 테마 |

#### 4-2-2. 채굴 프로세스

```
[광물 선택]
   ↓
[자동 채굴 시작] (서버에서 시뮬레이션)
   ↓ 매 1초마다
[HP 감소] HP -= DPS
   ↓ 클라이언트
[데미지 텍스트 표시] "-1,234"
[HP 바 애니메이션]
   ↓ HP ≤ 0
[채굴 완료!]
   ↓
[골드 지급 + 통계 업데이트]
   ↓
[5초 재등장 타이머 시작]
   ↓
[타이머 종료 → 광물 재선택 가능]
```

**재등장 시스템의 역할**:
- 상위 광물의 낮은 효율을 보완
- 대기 시간이 있어도 상위 광물이 더 유리하도록 설계

#### 4-2-3. 추천 시스템

**목적**: 현재 곡괭이 스펙으로 어떤 광물이 효율적인지 자동 판단

**효율 계산 공식**:
```python
채굴_시간 = 광물_HP / 유저_DPS
사이클_시간 = 채굴_시간 + 재등장_시간(5초)
골드_per_초 = 광물_골드 / 사이클_시간
```

**색상 표시**:
- 🟢 **초록 (Green)**: 추천 (효율 최고, 채굴 시간 1~5초)
- 🟡 **노랑 (Yellow)**: 도전 가능 (채굴 시간 5~30초)
- 🔴 **빨강 (Red)**: 비추천 (채굴 시간 30초 이상 or 매우 비효율)

**UX**:
- 광물 리스트에 색상 테두리
- "추천 광물 자동 선택" 버튼 제공

### 4-3. 오프라인 보상 시스템

#### 4-3-1. 기본 정책

| 항목 | 값 |
|------|------|
| **기본 시간** | 3시간 (KST 00:00 초기화) |
| **효율** | 50% (접속 시 대비) |
| **최대 누적** | 12시간 |
| **리셋 시간** | 매일 KST 00:00 |

> **Reset policy**: 모든 오프라인 타이머는 KST 00:00 기준으로 초기화되며, 해당 시점에 서버 세션이 없는 플레이어는 재접속 시 지연(Lazy) 정산 후 보상이 계산된다.

#### 4-3-2. 시간 확장 방법

| 방법 | 확장량 | 제한 |
|------|--------|------|
| **광고 시청** | +2시간 | 1일 3회 |
| **일일 미션** | 3개: +1h<br>5개: +1h<br>7개: +1h | 총 +3시간 |

**최대 시나리오**:
```
기본 3시간
+ 광고 3회 (6시간)
+ 일일 미션 7개 완료 (3시간)
= 총 12시간
```

#### 4-3-3. 서버 처리 방식

**방식**: 재접속 시 일괄 계산 (옵션 B)

**계산 로직**:
```python
# 유저 로그아웃 시
save_logout_time(user_id, server_timestamp)
save_current_mineral(user_id, mineral_id)

# 유저 재접속 시
delta_time = min(
    server_timestamp - logout_time,
    max_offline_time  # 최대 12시간
)

# 시뮬레이션
cycles = delta_time / (mineral.hp / user.dps)
gold = cycles × mineral.gold × 0.5  # 50% 효율

# 보상 지급
give_gold(user_id, gold)
```

> **Tutorial gate**: FTUE에서 첫 광물 선택이 끝나기 전까지는 오프라인 보상을 계산하지 않는다.

**게임 종료 시 모달**:

├─────────────────────────────┤

### 4-5. 곡괭이 슬롯 시스템

#### 4-5-1. 개요

**개념**: 여러 곡괭이를 동시에 배치하여 DPS 합산

**레이아웃**: 메인 화면 상단 1행 × 4칸 (Grid)

```
메인 화면:
┌─────────────────────────────────┐
│ [⛏️ Lv12] [⛏️ Lv5] [🔒] [🔒]  │ ← 슬롯
└─────────────────────────────────┘
```

#### 4-5-2. 슬롯 해금

| 슬롯 | 크리스탈 비용 | 무과금 기준 소요 일수 |
|------|-------------|---------------------|
| 슬롯 1 | - (기본) | - |
| 슬롯 2 | 400 | 3일 |
| 슬롯 3 | 2,000 | 15일 |
| 슬롯 4 | 4,000 | 30일 |

**해금 위치**:
- 메인 화면: 슬롯 직접 탭 → 해금 팝업
- 상점: 슬롯 해금 섹션

**연동**: 두 곳에서 해금 가능, 상태 공유

#### 4-5-3. 효과

**DPS 합산**:
```
슬롯 1: Lv 12 곡괭이 (DPS 1,638)
슬롯 2: Lv 5 곡괭이 (DPS 110)
---
총 DPS = 1,638 + 110 = 1,748
```

**추후 확장**:
- 곡괭이별 특수 효과 (속성, 크리티컬 보너스 등)
- 시너지 시스템 (특정 조합 시 보너스)
